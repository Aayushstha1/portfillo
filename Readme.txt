## 🌟 Project Overview

This is a **modern, responsive, and interactive portfolio** designed to showcase my:

- Skills & tools in web and mobile development
- Recent projects and accomplishments
- Personal introduction and experience
- Contact information for collaboration or hiring

The portfolio features smooth animations, a typing effect, and a clean modern UI/UX design.

---

## 🛠 Features

- **Hero Section**: Animated typing effect introducing me as a BCA student, developer, and learner.
- **About Section**: A brief personal introduction with image and description.
- **Skills Section**: Displays all technical skills with interactive hover effects.
- **Projects Section**: Highlights recent projects with description cards.
- **Responsive Design**: Works on all devices including mobile, tablet, and desktop.
- **Contact & Footer**: Simple contact info and social media links.
